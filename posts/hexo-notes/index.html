Hexo 官方文档

环境

  • 操作系统：Windows 11
  • Node 14.21.0
  • Git 2.39.1
  • Github

安装hexo

输入hexo -v验证是否安装成功

初始化hexo

  1. 在目标路径打开Git Bash：
  1. 进入项目文件夹
  • node_modules: 依赖包
  • public：存放生成的页面
  • scaffolds：生成文章的一些模板
  • source：用来存放你的文章
  • themes：主题
  • _config.yml: 博客的配置文件
  1. 创建新文章
  1. 启动项目
    输入hexo s 或者hexo server
  2. 浏览器打开http://localhost:4000/

Git+Github

环境配置

  1. 常用命令
  1. 配置用户名和邮箱
  1. 检查是否配置成功

连接Github

  1. 生成ssh公钥

一路回车，一般不需要设置密码。
找到.ssh文件夹，复制id_rsa.pub里的内容。
2. 配置到Github
settings→SSH and GPG keys→New SSH key
查看是否成功

将hexo部署到GitHub

  1. 安装deploy-git
  1. 打开站点配置文件 _config.yml，修改最后一行。官方文档
  1. 运行命令
  • hexo clean/hexo cl：清除缓存
  • hexo generate/hexo g：生成静态文章
  • hexo deploy/hexo d：部署文章
    如果出现Deploy done，则说明部署成功了。
